

# Networking Basics

## What is Networking?

Networking connects computers and devices so they can communicate and share information.

## OSI Model

1. Physical Layer
2. Data Link Layer
3. Network Layer
4. Transport Layer
5. Session Layer
6. Presentation Layer
7. Application Layer

## Common Protocols

### HTTP
Used for websites.

### HTTPS
Secure version of HTTP.

### TCP
Reliable data transfer.

### UDP
Fast data transfer.

### DNS
Converts domain names into IP addresses.

### SSH
Used for secure remote access.

## IP Address

An IP address identifies a device on a network.

Example:
192.168.1.1

## Why Networking Matters

- Helps understand cyber attacks.
- Required for Nmap and Wireshark.
- Important for network security.

## Summary

Networking is one of the most important foundations of cybersecurity.
