

# Linux Basics

## What is Linux?
Linux is an open-source operating system widely used in servers, cloud computing, and cybersecurity.

## Important Commands

### pwd
Shows the current directory.

### ls
Lists files and folders.

### cd
Changes the current directory.

### mkdir
Creates a new folder.

### rm
Removes files or folders.

### cp
Copies files.

### mv
Moves or renames files.

### cat
Displays file contents.

### grep
Searches for text in files.

## File Permissions

### chmod
Changes file permissions.

### chown
Changes file ownership.

## Why Linux is Important in Cybersecurity

- Most servers run Linux.
- Many security tools work best on Linux.
- Ethical hackers commonly use Linux distributions such as Kali Linux.
