# Authentication vs Authorization

## What is Authentication?

Authentication is the process of verifying a user's identity.

Examples:
- Logging in with a username and password
- Using a fingerprint
- Using a one-time password (OTP)

## What is Authorization?

Authorization determines what an authenticated user is allowed to access.

Examples:
- An admin can manage users.
- A regular user can only view their own account.

## Key Difference

Authentication answers:
"Who are you?"

Authorization answers:
"What are you allowed to do?"

## Real-World Example

When you log into a website:
1. Authentication verifies your identity.
2. Authorization determines which features and data you can access.

## Conclusion

Authentication and authorization work together to keep systems secure.
