# Web Security Basics

## What is Web Security?

Web security is the practice of protecting websites and web applications from cyber attacks.

## Common Threats

### SQL Injection
An attacker inserts malicious SQL commands into input fields.

### Cross-Site Scripting (XSS)
An attacker injects malicious scripts into web pages.

### Cross-Site Request Forgery (CSRF)
Tricks a user into performing unwanted actions.

### Weak Passwords
Weak passwords can be easily guessed or cracked.

## Security Best Practices

- Use strong passwords
- Enable two-factor authentication
- Keep software updated
- Validate user input
- Use HTTPS

## Why Web Security Matters

Most modern services are web-based, making web security an important cybersecurity skill.
