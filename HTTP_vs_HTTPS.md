# HTTP vs HTTPS

## What is HTTP?

HTTP (HyperText Transfer Protocol) is used to transfer data between a web browser and a web server.

## What is HTTPS?

HTTPS (HyperText Transfer Protocol Secure) is the secure version of HTTP. It uses SSL/TLS encryption to protect data.

## Key Differences

- HTTP does not encrypt data.
- HTTPS encrypts data.
- HTTP uses port 80.
- HTTPS uses port 443.
- HTTPS requires an SSL/TLS certificate.

## Why HTTPS is Important

- Protects user data.
- Prevents attackers from reading information in transit.
- Increases trust and security.

## Conclusion

HTTPS is safer than HTTP and should be used whenever sensitive information is transmitted.
