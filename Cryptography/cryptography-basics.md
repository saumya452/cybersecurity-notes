# Cryptography Basics

## What is Cryptography?

Cryptography is the science of protecting information using mathematical techniques.

## Key Concepts

### Encryption
Converts readable data into unreadable data.

### Decryption
Converts encrypted data back into readable form.

### Cipher
An algorithm used for encryption and decryption.

## Types of Encryption

### Symmetric Encryption
Uses the same key for encryption and decryption.

Examples:
- AES
- DES

### Asymmetric Encryption
Uses a public key and a private key.

Examples:
- RSA
- ECC

## Why Cryptography Matters

- Protects sensitive information
- Secures online banking
- Secures websites using HTTPS
- Ensures privacy and authenticity

## Summary

Cryptography is a fundamental part of modern cybersecurity.
