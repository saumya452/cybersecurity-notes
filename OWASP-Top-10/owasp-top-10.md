# OWASP Top 10 Notes

## SQL Injection (SQLi)
A vulnerability where attackers insert malicious SQL commands into input fields.

## Cross-Site Scripting (XSS)
A vulnerability that allows attackers to inject malicious scripts into web pages.

## Broken Authentication
Weak authentication mechanisms that allow attackers to gain unauthorized access.

## Security Misconfiguration
Improper security settings that expose applications to attacks
